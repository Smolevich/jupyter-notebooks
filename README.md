# Jupyter Notebooks & HH.ru API Tools

Репозиторий содержит набор Jupyter-ноутбуков и вспомогательных Python-скриптов для автоматизации работы с API HeadHunter (HH.ru) и анализа данных.

## Основные инструменты

### Jupyter Notebooks
*   **`hh_api_loader.ipynb`** — загрузка вакансий из HH.ru по заданным категориям и сохранение их в ClickHouse. Поддерживает динамическую фильтрацию по ролям и категориям из `search_config.json`.
*   **`hh_api_updating.ipynb`** — обновление данных о вакансиях в ClickHouse (синхронизация статусов архивации, удаления и обновление описаний).
*   **`hh_data_visualizer.ipynb`** — визуализация и анализ собранных данных о вакансиях.
*   **`telegram_stats.ipynb`** — анализ статистики Telegram-каналов.

### Python Скрипты
*   **`hh_api_tool.py`** — интерактивная утилита для быстрого поиска в HH API. Позволяет получать детализацию вакансий и профили работодателей в JSON-формате прямо в терминале.
*   **`check_hh_api.py`** — скрипт для проверки доступности API и валидности токенов.

## Конфигурация
Для работы инструментов требуются следующие файлы в корне проекта (не отслеживаются в Git):
*   `config.json` — параметры подключения к API и базе данных.
*   `token.json` — кэшированные токены доступа.
*   `search_config.json` — конфигурация поисковых запросов и категорий.

## Принципы разработки
*   **Clean Code**: Минимум комментариев, самодокументированный код.
*   **No Outputs in Git**: Ноутбуки сохраняются без результатов выполнения ячеек.
*   **Minimal Dependencies**: Предпочтение стандартной библиотеке Python.
