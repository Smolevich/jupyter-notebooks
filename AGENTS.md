# Project Repository

This repository is dedicated to managing **Jupyter Notebooks** and auxiliary **Python scripts** for data analysis and automation.

### Development Principles
- **Clean Code**: Prioritize clarity and conciseness in Python code.
- **Standard Library**: Prefer `stdlib` whenever possible. Use external libraries only when necessary.
- **Minimal Comments**: Aim for self-documenting code. Use comments only for complex logic that cannot be simplified.

### Project Structure
- **Notebooks**: Analytical tools for HH.ru API integration and automation tasks.
- **Scripts**: Maintenance and environment verification utilities.

Detailed information about specific tools and usage can be found in the [Runbooks](runbooks/README.md).
