# Runbooks & Details

### Tool Usage

#### HH.ru API Check
- `check_hh_api.py`: A script for rapid verification of HH.ru API connectivity and app token permissions (Scopes).
- **Usage**: `python3 check_hh_api.py`
- **Config**: Relies on `app_token` and `user_email` in `config.json`.

### Project Components
- **Jupyter Notebooks**: Core data analysis and automation tools (e.g., `hh_api_updating.ipynb`).
- **Python Scripts**: Utility scripts for environment validation and background tasks.

### Configuration
- `config.json`: Centralized storage for API keys, database credentials, and application tokens.
- `requirements.txt`: Python dependency list.
